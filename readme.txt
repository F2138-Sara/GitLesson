

###GIT SETUP###

Open CMD Command terminal on windows
update your name and email. 
use the following code one by one 

-----------
git config --global user.name "your name"
git config --global user.email "your@email.com"
----------

Then 

git clone https://github.com/selcukgoktas/GitLesson.git

then create a folder with your name and add some file into this folder. 

then pull the first after that push your files to this repository.
